<i>SD Maid 2/SE</i> 是你 Android 裝置的可信賴助手，讓它保持整潔有序。

沒有人是完美的，Android 也不例外。

* 你已移除的應用程式仍留有殘留資料。
* 日誌、崩潰報告及其他你不需要的檔案不斷被建立。
* 你的儲存空間正在積累你無法辨識的檔案和資料夾。
* 你相簿中的重複相片。

不多說了… 讓 SD Maid 2/SE 來幫你吧！

SD Maid 2/SE 是一款應用程式及檔案管理器，專門識別裝置上哪些應用程式建立了特定檔案。 SD Maid 2/SE 會掃描你的裝置，並將檔案與已安裝的應用程式進行比對，提供安全釋放儲存空間的選項。

✨ 解除安裝應用程式後進行清理

如果應用程式在指定資料夾以外建立檔案，即使解除安裝應用程式後，這些檔案仍可能保留。 「CorpseFinder」工具可找出應用程式殘留資料，告訴你它們屬於哪個應用程式，並幫助你刪除它們。

🔍 以智慧方式搜尋你的裝置

篩選空資料夾、臨時檔案、已安裝的應用程式等等。 你甚至可以建立自己的搜尋條件。 「SystemCleaner」工具可讓你根據各種條件自動搜尋裝置並刪除檔案。

🧹 刪除可消耗的檔案和隱藏的快取

縮圖、垃圾桶、離線快取等等：如果應用程式不自行清理，這款應用程式會代勞。 「AppCleaner」工具可找出含有可消耗檔案的應用程式。

📦 管理你所有的應用程式

取得裝置上所有已安裝應用程式的完整清單。 已啟用、已停用、使用者或系統應用程式：沒有應用程式能躺過你的眼睛。 「AppControl」工具是一款應用程式管理器，可讓你搜尋、排序和控制你的應用程式。

📊 什麼佔用了你的所有空間

儲存空間管理可能很複雜，手機儲存、SD 卡和 USB 裝置上分佈著應用程式、媒體、系統及其他檔案。 「StorageAnalyzer」是一款檔案管理器，可顯示裝置上空間的使用情況，簡化你的儲存空間管理。

📷 尋找重複資料

重複的下載內容、透過社交媒體傳送的相片，或只是同一場景的相似圖片：隨著時間推移，副本可能會越積越多。 「Deduplicator」工具可找出完全相同或相似的檔案，幫助你刪除多餘的副本。

👆 滑動整理雜亂

選擇一個資料夾，透過簡單的滑動逐一查看檔案，以保留、刪除或略過。 「Swiper」工具可讓你在刪除任何內容之前審查所有決定，並且你可以儲存工作階段以便日後繼續。

🗃️ 不刪除相片節省空間

相片可能佔用大量儲存空間，但你不一定想要刪除它們。 「Media Squeeze」工具可使用可調整的品質設定壓縮你的圖片，在保留相片的同時釋放空間。

此應用程式無廣告。 部分功能需要付費升級。

SD Maid 2/SE 是 SD Maid 1/Legacy 的繼任版本。
針對較新的 Android 版本進行最佳化，專注於清理工作。

此應用程式具有可選功能，可利用 AccessibilityService API 自動化繁琐的操作。
透過使用 AccessibilityService API，此應用程式可代替你點擊按鈕，對多個應用程式執行操作，例如 正在刪除快取。
此應用程式不會使用 AccessibilityService API 來收集資訊。

SD Maid 2/SE 是一款檔案管理器及清理應用程式。
